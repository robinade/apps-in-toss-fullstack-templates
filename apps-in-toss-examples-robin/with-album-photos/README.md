# with-album-photos

앨범에서 다중 이미지를 선택하는 AppsInToss SDK 2.0.1 예제입니다.

## SDK API

- **`fetchAlbumPhotos({ maxCount })`** - 앨범에서 사진을 선택합니다
- **`fetchAlbumPhotos.isSupported()`** - 앨범 접근 지원 여부를 확인합니다

## Features

- 다중 사진 선택 (최대 5장)
- 3열 그리드 미리보기
- 개별 사진 삭제
- 전체 삭제
- 웹 환경 폴백 (file input)

## Run

```bash
npm install
npm run dev
```
